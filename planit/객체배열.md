### 객체 배열

- 참조변수 배열이라고도 부름

```java
Tv t1 = new Tv();
Tv t2 = new Tv();
Tv t3 = new Tv();

Tv[] tv_arr = new Tv[3];
tv_arr[0] = new tv();
tv_arr[1] = new tv();
tv_arr[2] = new tv();

tv_arr[] = { new Tv(), new Tv(), new Tv()};
```
